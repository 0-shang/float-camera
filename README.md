# FloatCamera 📸

[English](./README.md) | [简体中文](./README_ZH.md)

FloatCamera is a professional-grade floating camera application for Windows. It provides an "Always-on-Top" camera window with real-time AI-powered features like portrait beautification and background removal/replacement.

## ✨ Features

- **Floating Window**: Always stays on top of other applications.
- **AI Background Removal**: Powered by TensorFlow.js and MediaPipe for smooth background replacement.
- **Portrait Beautification**: Real-time filters for brightness, contrast, and saturation.
- **Flexible Shapes**: Toggle between circular and rectangular camera views.
- **Customizable**: Adjust opacity, scale, and window positioning easily.
- **Windows Optimized**: Built specifically for a seamless Windows desktop experience.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/float-camera.git
   cd float-camera
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```
