# FloatCamera 📸

[English](./README.md) | [简体中文](./README_ZH.md)

FloatCamera 是一款专为 Windows 设计的专业级浮动摄像头应用。它提供了一个“始终置顶”的摄像头窗口，并集成了基于 AI 的实时人像美颜和背景移除/替换功能。

## ✨ 功能特性

- **浮动窗口**：始终保持在其他应用程序之上。
- **AI 背景移除**：采用 TensorFlow.js 和 MediaPipe 技术，实现平滑的背景替换。
- **人像美颜**：支持亮度、对比度和饱和度的实时滤镜调节。
- **多种形状**：可在圆形和矩形摄像头视图之间自由切换。
- **高度可定制**：轻松调整不透明度、缩放比例和窗口位置。
- **Windows 优化**：专为 Windows 桌面体验打造。

## 🚀 快速开始

### 前置条件

- [Node.js](https://nodejs.org/) (建议使用 v16 或更高版本)
- [npm](https://www.npmjs.com/)

### 安装步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/YOUR_USERNAME/float-camera.git
   cd float-camera
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动应用：
   ```bash
   npm start
   ```

## 🛠️ 技术栈

- [Electron](https://www.electronjs.org/) - 桌面应用框架
- [TensorFlow.js](https://www.tensorflow.org/js) - 浏览器端的机器学习
- [MediaPipe](https://mediapipe.dev/) - 背景分割技术
